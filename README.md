# Oybek Abdullaev — Personal Website

## How to put this on the internet (GitHub Pages)

You already have a repo called `oybekabout.github.io`. Just follow these steps exactly.

### Step 1 — Empty out the repo
Go to your repo on github.com. For every file currently listed (README.md, index.html, photo-1.jpg, script.js, style.css, etc.):
- Click on the file name
- Click the trash-can icon in the top right of the file view
- Click "Commit changes" at the bottom

Repeat until the repo is empty. (This takes a couple of minutes — that's OK.)

### Step 2 — Upload everything in this folder, all at once
1. On your repo page, click the green **Add file** button → **Upload files**
2. Open the folder on your computer where you unzipped this download
3. Select ALL the files inside it (index.html, style.css, script.js, and all the photo-1.jpg through photo-8.jpg files) — click the first one, then Shift+click the last one to select them all at once
4. Drag all of them into the GitHub upload box (or use "choose your files")
5. Scroll down, type any short message like "upload site", and click the green **Commit changes** button

That's it — no folders to create, no renaming, nothing else.

### Step 3 — View your site
Wait about 1 minute, then visit:

**https://oybekabout.github.io/**

If it still looks broken, do a hard refresh: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac).

## Editing later
- Text: open `index.html`, find the words you want to change, edit, commit.
- Colors/fonts: open `style.css`, the very top section lists all colors.
